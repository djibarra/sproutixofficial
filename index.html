About Sproutix

Sproutix (SPX) is a utility token built on the TON network, designed to power a casual farming‑production mini‑game fully integrated with Telegram.

The goal is to create a simple, fun, and rewarding game economy while growing an active Web3 community.

About the Game

Sproutix is a farm‑production game where players plant, grow, harvest and upgrade their farm. Everything is designed to be fast, simple, and ideal for Mini App users.

SPX is used inside the game to unlock crops, speed up timers, buy tools, and access special events.

SPX Token Utility

  • Upgrade farm buildings
  • Buy seeds and tools
  • Speed up crop production
  • Trade valuable resources
  • Unlock special farming zones
  • Join limited‑time events
